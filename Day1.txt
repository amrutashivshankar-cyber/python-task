##Data types

#This example is based on all the predefined data types and how to use it. 

Program Code:

name = 'Prashant Jha'    
marks = 80    
percentage = 93.5    
complex_value= 2+8j    
passed     = True    
print('Display the output:')    
print('Name of Person:', name)    
print('Marks                 :', marks)    
print('Percentage       :', percentage)    
print('Complex value :', complex_value)    
print('Passed status   :', passed)    
Output 

Name of Person: Prashant Jha 
Marks : 80 
Percentage : 93.5 
Complex value : (2+8j) 
Passed status : True

#WAP to check the Datatype of Variable 

Program Code:

marks  = 80  
percentage = 93.5  
Complex_value = 2+8j  
passed    = True  
print('Display the output:')  
print('Name (Data type=)               :', type(name))  
print('Marks(Data type=)                :', type(marks))  
print('Percentage(Data type=)       :', type(percentage))  
print('Complex value(Data type=) :', type(complex_value))  
print('Passed status(Data type=)   :', type(passed))

##WAP to check if three variables have the same value so all the three variable addresses are the same. ( Using id() function we can check the address) 

Program Code:


math = 50  
chemistry = 50  
Physics = 50  
print('Address of Math      =:', id(math))  
print('Address of chemistry =:', id(chemistry))  
print('Address of physics   =:', id(physics))  
Program Output:

Address of Math =: 140708003827672
Address of chemistry =: 140708003827672
Address of physics =: 140708003827672
